Games as Laboratories for Visual Cognition Research

Curriculum development · Perception in Virtual Worlds · In development

I helped develop Perception in Virtual Worlds, a curriculum using game environments as a platform for perception research and teaching experimental design. Below are case studies — not completed experiments, but examples of how classic arcade games can isolate perceptual variables in dynamic, embodied contexts.

Case Studies

Tailed Pac-Man

Can a single visual appendage change how vulnerable an agent appears — and how long it survives? I added a simple "tail" to Pac-Man to test whether minimal visual structure alters perceived vulnerability. The tailed agent was caught faster and elicited more abrupt movement patterns from the ghost AI.

▶ Play Tailed Pac-Man

Color Pong

Does color alter not only how a game feels — but how we play it? Identical Pong environments differing only in background color. Color influenced time spent playing, movement patterns, and performance. Red environments produced faster, more erratic paddle movement.

▶ Play Color Pong

Maze Width Illusion

If two paths are the same length, can width alone determine which one we choose? In a virtual maze, participants chose between identical-length routes that differed only in width. Wider paths were perceived as shorter and chosen significantly more often — a spatial judgment bias with real navigational consequences.
